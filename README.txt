GETTING STARTED
───────────────
1. Extract this entire folder to a permanent location
   (e.g., C:\Program Files\WealthWatch or your Desktop)

2. Double-click WealthWatch.exe to launch the app

3. Your browser will open automatically to the WealthWatch
   login screen

4. On first launch, a PIN will be generated for you — write
   it down! You'll need it each time you open the app

5. Click "Desktop Shortcut" on the login page to create a
   shortcut for easy access


IMPORTANT
─────────
- Do NOT run WealthWatch.exe from inside the zip file.
  You must extract the folder first.

- Keep all files together. WealthWatch.exe needs the
  _internal folder next to it to run.

- Your data is stored separately in:
  %LOCALAPPDATA%\WealthWatch\
  So you can update WealthWatch without losing your data.


CLOSING THE APP
───────────────
Just close the browser tab. The background server will
shut down automatically after a few minutes.


FORGOT YOUR PIN?
────────────────
Delete this file and restart the app to get a new PIN:
%LOCALAPPDATA%\WealthWatch\.pin

(Type %LOCALAPPDATA%\WealthWatch in File Explorer's
address bar to find it)
