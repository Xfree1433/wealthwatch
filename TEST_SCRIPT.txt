===============================================================================
  WEALTHWATCH - User Acceptance Test Script
  Personal Finance Dashboard (Desktop App)
  URL: http://127.0.0.1:5100 (auto-opens in browser)
  Login: PIN is auto-generated on first run and displayed on the login page
===============================================================================

HOW TO START:
  cd WEALTHWATCH
  venv\Scripts\activate
  python launcher.py
  (Browser opens automatically)

  FIRST RUN: The app generates a PIN and displays it on the login page.
  Write it down! PIN is stored at: %LOCALAPPDATA%\WealthWatch\.pin
